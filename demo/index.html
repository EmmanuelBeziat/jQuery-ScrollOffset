Qu'est-ce que c'est quoi, dis donc ?

ScrollOffset, c'est un petit plugin jQuery tout simple, qui permet de rendre la navigation de vos ancres toute fluide comme l'eau de source d'une publicité mal inspirée, et ajouter un décalage vertical si l'envie vous en prend.

What is this thing, anyway ?

ScrollOffset is a simple and light jQuery plugin, that make your anchor navigation really smooooth like the clear and fresh water of an uninspired advertising, and add a vertical offset if you want.

Comment ça marche ?

Le principe est basé sur les ancres html (non, c'est vrai ?). Appelez simplement la méthode scrollOffset() sur un objet jQuery, qui soit un lien html comportant une ancre.

$('a[href^="#"]').scrollOffset();

How does it work ?

The principle is based on html anchors (no kidding ?). Just call for the method scrollOffset() on a jQuery object that is actually a html link with an anchor.

$('a[href^="#"]').scrollOffset();
